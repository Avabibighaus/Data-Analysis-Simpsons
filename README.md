# Simpsons Data Analysis Project

## Overview

This project was completed as a collaborative software development course project at California Polytechnic State University.

Our team analyzed multiple public datasets related to *The Simpsons*, including character, episode, and ratings data. We synthesized information from these datasets to identify trends and patterns, then presented our findings through a website featuring interactive data visualizations.

The goal of the project was to transform raw data into meaningful insights that could be easily explored and understood by users.
